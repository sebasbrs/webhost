Server requirements (important!)

What do I need to run this system?

• /application/config/ needs to be writable (see chmod)
• /application/modules/ is writable
• /application/cache/
If you're on a web host, Apache may already be properly configured.
• Apache: mod_rewrite, mod_headers, mod_expires, mod_deflate
Go into your Apache directory and find the httpd.conf file. Mine was located in "C:\wamp\bin\apache\apache2.2.22\conf\". Open the file with a text editor and search (CTRL+F) for one of the modules you need to enable. To enable them, simply remove the #-character in front of the line.

• PHP: php_curl, php_openssl, php_soap, php_gd, php_mbstring, php_json
Go into your PHP directory and find the php.ini file. Mine was located in "C:\wamp\bin\php\php5.3.13". Open the file with a text editor and search (CTRL+F) for one of the modules you need to enable. To enable them, simply remove the ;-character in front of the line.
